worlo Teams for your org

worlo Teams is a separate app from Microsoft Teams. Your admin generates an invite code in the dashboard; teammates install the app, paste the code, and chat with the agent right away.

  1. Sign in to the worlo dashboard and open worlo Teams
  2. Generate an invite code for the agent you want to share
  3. Open worlo Teams, enter your name and code, then start chatting
